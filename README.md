# KIFT
### USAGE


1. Clone and and enter repository:
```
git clone https://github.com/pstringe/KIFT
cd KIFT
```

2. Install dependencies:
```
make setup
```

3. Start server:
```
./server [availaible port number]
```

4. Start client:
```
./client [server port number]
```

  Or in debug mode:

```
./client --debug [server port number]
```


### Available Commands

* WHOIS
* WHEREIS
* BRIGHT
* DIM
* QUIT
* HISTORY
* PLAYMUSIC
* SETALARM
* SEARCHTHEWEB
* SENDEMAIL
* SENDSMS
* TEAM
* WEATHER
